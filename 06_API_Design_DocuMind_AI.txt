Authentication EndpointsPOST /auth/registerRegister a new user account.Auth Required: NoRequest Body: { &quot;email&quot;: &quot;string&quot;, &quot;password&quot;: &quot;string&quot;, &quot;full_name&quot;: &quot;string&quot; }Response: { &quot;id&quot;: &quot;uuid&quot;, &quot;email&quot;: &quot;string&quot;, &quot;full_name&quot;: &quot;string&quot; }Status Codes: 201 Created, 400 Bad Request, 409 ConflictPOST /auth/loginAuthenticate and receive a JWT token.Auth Required: NoRequest Body: { &quot;email&quot;: &quot;string&quot;, &quot;password&quot;: &quot;string&quot; }Response: { &quot;access_token&quot;: &quot;string&quot;, &quot;token_type&quot;: &quot;bearer&quot; }Status Codes: 200 OK, 401 UnauthorizedPOST /auth/logoutEnd the current user session.Auth Required: YesRequest Body: NoneResponse: { &quot;message&quot;: &quot;Logged out successfully&quot; }Status Codes: 200 OK, 401 Unauthorized3. Project EndpointsGET /projectsRetrieve all projects for the authenticated user.Auth Required: YesRequest Body: NoneResponse: { &quot;projects&quot;: [ { &quot;id&quot;: &quot;uuid&quot;, &quot;name&quot;: &quot;string&quot;, &quot;industry&quot;: &quot;string&quot;, &quot;created_at&quot;: &quot;datetime&quot; } ] }Status Codes: 200 OK, 401 UnauthorizedPOST /projectsCreate a new project.Auth Required: YesRequest Body: { &quot;name&quot;: &quot;string&quot;, &quot;industry&quot;: &quot;string&quot;, &quot;description&quot;: &quot;string&quot; }Response: { &quot;id&quot;: &quot;uuid&quot;, &quot;name&quot;: &quot;string&quot;, &quot;created_at&quot;: &quot;datetime&quot; }Status Codes: 201 Created, 400 Bad RequestGET /projects/{project_id}Get full details of a specific project.Auth Required: YesRequest Body: NoneResponse: { &quot;id&quot;: &quot;uuid&quot;, &quot;name&quot;: &quot;string&quot;, &quot;industry&quot;: &quot;string&quot;, &quot;description&quot;: &quot;string&quot;, &quot;reports&quot;: [...] }Status Codes: 200 OK, 404 Not FoundDELETE /projects/{project_id}Permanently delete a project and all its data.Auth Required: YesRequest Body: NoneResponse: { &quot;message&quot;: &quot;Project deleted successfully&quot; }Status Codes: 200 OK, 404 Not Found4. AI Analysis EndpointsPOST /projects/{project_id}/analyzeStart an AI intake session. AI returns initial follow-up questions.Auth Required: YesRequest Body: { &quot;business_description&quot;: &quot;string&quot; }Response: { &quot;session_id&quot;: &quot;uuid&quot;, &quot;questions&quot;: [&quot;string&quot;] }Status Codes: 200 OK, 400 Bad RequestPOST /projects/{project_id}/questionsSubmit answers and receive next questions or final analysis.Auth Required: YesRequest Body: { &quot;session_id&quot;: &quot;uuid&quot;, &quot;answers&quot;: [{&quot;question&quot;: &quot;string&quot;, &quot;answer&quot;: &quot;string&quot;}] }Response: { &quot;status&quot;: &quot;in_progress | completed&quot;, &quot;next_questions&quot;: [&quot;string&quot;], &quot;analysis&quot;: { ... } }Status Codes: 200 OK, 400 Bad Request5. Document Generation EndpointsGET /projects/{project_id}/reportsList all generated reports for a project.Auth Required: YesRequest Body: NoneResponse: { &quot;reports&quot;: [{ &quot;id&quot;: &quot;uuid&quot;, &quot;report_type&quot;: &quot;string&quot;, &quot;created_at&quot;: &quot;datetime&quot; }] }Status Codes: 200 OK, 404 Not FoundPOST /projects/{project_id}/generateGenerate one or more selected documents.Auth Required: YesRequest Body: { &quot;document_types&quot;: [&quot;BRD&quot;, &quot;SRS&quot;, &quot;SWOT&quot;, &quot;UserStories&quot;, &quot;TestCases&quot;] }Response: { &quot;reports&quot;: [{ &quot;id&quot;: &quot;uuid&quot;, &quot;report_type&quot;: &quot;string&quot;, &quot;preview_url&quot;: &quot;string&quot; }] }Status Codes: 202 Accepted, 400 Bad RequestGET /reports/{report_id}/downloadDownload a report in the specified format.Auth Required: YesRequest Body: Query param: ?format=pdf|docx|pptx|xlsxResponse: File stream (binary)Status Codes: 200 OK, 404 Not Found, 400 Bad Request6. AI Chat EndpointPOST /reports/{report_id}/chatSend a message to the AI in the context of a specific report.Auth Required: YesRequest Body: { &quot;message&quot;: &quot;string&quot;, &quot;history&quot;: [{&quot;role&quot;: &quot;user|assistant&quot;, &quot;content&quot;: &quot;string&quot;}] }Response: { &quot;response&quot;: &quot;string&quot; }Status Codes: 200 OK, 404 Not Found7.
